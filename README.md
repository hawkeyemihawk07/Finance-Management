# 📂 Document Management System

A **Document Management System (DMS)** is a web-based application developed to manage, store, and organize important digital documents securely. This project aims to reduce paperwork, improve document accessibility, and provide users with a centralized platform for managing their files efficiently.

---

## 🚀 Features

✅ User Registration & Login Authentication
✅ Upload Documents Securely
✅ View Stored Documents
✅ Edit Document Details
✅ Delete Documents
✅ Search Documents Easily
✅ Categorization of Files
✅ Secure Data Management
✅ Simple and User-Friendly Interface

---

## 🎯 Objectives

* To digitize document storage and management
* To reduce manual paperwork
* To provide secure document access
* To improve efficiency in retrieving documents
* To organize documents systematically
* To ensure easy maintenance and updates

## ⚙️ Installation Guide

### Step 1: Clone the Repository

```bash
git clone https://github.com/hawkeyemihawk07/Document-Management-System.git
```

### Step 2: Navigate into the Project Folder

```bash
cd Document-Management-System
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run the Application

```bash
python app.py
```

### Step 5: Open in Browser

```bash
http://127.0.0.1:5000/
```

## 💡 How It Works

1. User logs into the system
2. Uploads important documents
3. Documents are stored in the database/server
4. User can search, update, or delete documents
5. Documents remain accessible anytime
